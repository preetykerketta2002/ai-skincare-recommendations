# AI Based Facial Skin Analyzing System

An AI-powered Facial Skin Analysis and Skincare Recommendation System developed using Deep Learning and Transfer Learning techniques.  
The system analyzes facial skin conditions from images and provides skincare recommendations based on detected skin issues.

---

## Features

- Facial skin analysis using AI
- Skin type detection
- Acne and skin issue prediction
- Skincare product recommendations
- Image-based prediction system
- Deep Learning powered model
- User-friendly interface


---

## Class Description

### 1. SkinAnalyzer
Analyzes facial skin conditions from uploaded images.

### 2. ImageProcessor
Handles image preprocessing such as:
- Resizing
- Normalization
- Noise reduction
- Image enhancement

### 3. ModelTrainer
Trains the deep learning model using the dataset.

### 4. PredictionSystem
Predicts skin conditions using the trained AI model.

### 5. RecommendationSystem
Provides skincare recommendations based on prediction results.

### 6. DatasetLoader
Loads and manages dataset images for training and testing.
